Sobre o Projeto

Este projeto foi desenvolvido em grupo como atividade da disciplina de Programação em Microinformática, da FATEC Ipiranga.

O objetivo do projeto foi criar uma solução para sanar dores operacionais de uma empresa, por meio do desenvolvimento de um sistema de controle de estoque automatizado utilizando Excel, VBA e Macros.

A proposta consistiu em transformar processos manuais e suscetíveis a erros em um fluxo mais eficiente, automatizado e organizado.

🚀 Funcionalidades

O sistema contempla funcionalidades como:

Cadastro de produtos
Controle de entrada e saída de mercadorias
Atualização automática do estoque
Consulta de itens disponíveis
Registro e acompanhamento de movimentações
Alertas para baixo estoque (caso implementado)
Automação de tarefas com macros
🛠 Tecnologias Utilizadas
Microsoft Excel
VBA (Visual Basic for Applications)
Macros para automação
Lógica de programação
Estruturação de planilhas e dados

🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

Automatizar o controle de estoque
Reduzir erros manuais
Melhorar a organização dos dados
Aumentar a eficiência operacional
Aplicar conceitos aprendidos em sala de aula em um caso prático

📚 Contexto Acadêmico

Projeto desenvolvido como requisito da disciplina Programação em Microinformática, no curso de graduação da FATEC Ipiranga.

A atividade tinha como proposta identificar problemas reais de uma empresa e propor uma solução computacional utilizando ferramentas de automação em ambiente de microinformática.
